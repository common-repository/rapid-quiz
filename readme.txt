Rapid Quiz is the quickest, easiest way to create multiple choice questions, quizzes and exercises for WordPress.

== Description ==

A simple way to add multiple choice questions and answers to posts and pages, directly from the Edit Post / Edit Page screen, without needing to know programming code. 

You can see a live <a href="http://www.english-at-home.com/grammar-exercise-quantifiers/">demo here</a>.

Developed by <a href="http://www.rapidesl.com/">Rapid ESL</a>.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Take a look at the new menu option Rapid Quiz for instructions on how to add quiz questions.

== Frequently Asked Questions ==

= Can I see a live demo of Rapid Quiz?  =

Check out this example on our <a href="http://www.english-at-home.com/exercise-articles/">English as a Second Language site</a>.

= Can I add my own CSS styles to the quiz? =

Yes. After installing Rapid Quiz, choose Rapid Quiz > Settings from the admin menu for a list of CSS IDs and classes you can use in a custom stylesheet.
